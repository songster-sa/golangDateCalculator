Date calculator

Given 2 dates, in the format dd/MM/yyyy, this calculator returns the number of days elapsed between the dates, excluding both begin and end dates

Instructions to run :
$ go build
$ ./dateCalculator

Sample output :

Enter experiment beign date:
12/12/2000
Enter experiment end date:
01/01/2001
result :  19
error :  <nil>
